# 📖 ห้องสมุดออนไลน์ (Online Book Reader)

เว็บอ่านหนังสือออนไลน์แบบ static — หน้าแรกรวมรายการหนังสือ คลิกเข้าไปอ่านแต่ละเล่มได้ทันที
เปิดได้ทุกอุปกรณ์ (มือถือ/แท็บเล็ต/คอม) และโฮสต์ฟรีบน GitHub Pages

## 📚 หนังสือในห้องสมุด

| รหัสวิชา | ชื่อเรื่อง |
|----------|-----------|
| 96408 | ฐานข้อมูลคลังสินค้า บริษัท ABC — คู่มือ SQL (DDL/DML/SELECT) |
| 96408 | การจัดการระบบฐานข้อมูล (ER, นอร์มัลไลเซชัน) |
| 96304 | การสื่อสารข้อมูลและระบบเครือข่ายคอมพิวเตอร์ |
| 96404 | การตรวจสอบระบบงานคอมพิวเตอร์และการควบคุมภายใน |

## 🗂 โครงสร้างไฟล์

```
.
├── index.html              ← หน้าห้องสมุด (รวมรายการหนังสือ)
├── assets/
│   ├── catalog.js          ← รายการหนังสือ (แก้ที่นี่เวลาเพิ่มเล่ม)
│   └── library.css         ← สไตล์หน้าห้องสมุด
├── books/                  ← ไฟล์ HTML ของหนังสือแต่ละเล่ม
│   ├── sql-96408.html
│   ├── db-96408.html
│   ├── network-96304.html
│   └── audit-96404.html
└── .github/workflows/deploy-pages.yml   ← deploy อัตโนมัติขึ้น GitHub Pages
```

## ➕ วิธีเพิ่มหนังสือเล่มใหม่

1. วางไฟล์ `.html` ของหนังสือไว้ในโฟลเดอร์ `books/`
2. เปิด `assets/catalog.js` แล้วคัดลอกบล็อก `{ ... }` มา 1 อัน แก้ค่าให้ตรงเล่มใหม่
   (ชื่อเรื่อง, รหัสวิชา, ไอคอน, สีหน้าปก, พาธไฟล์ ฯลฯ)
3. commit + push — เว็บจะอัปเดตเองอัตโนมัติ

## 🚀 การเผยแพร่ (GitHub Pages)

มี workflow ตั้งค่า deploy อัตโนมัติให้แล้ว เปิดใช้งานครั้งแรกแค่ครั้งเดียว:

1. ไปที่ **Settings → Pages** ของ repository
2. ที่หัวข้อ **Build and deployment → Source** เลือก **GitHub Actions**
3. ทุกครั้งที่ push ขึ้น branch `main` เว็บจะถูก deploy ใหม่อัตโนมัติ
   URL จะเป็นประมาณ `https://<username>.github.io/<repo>/`

## 💻 ลองเปิดในเครื่อง

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย หรือรันเซิร์ฟเวอร์ง่าย ๆ:

```bash
python3 -m http.server 8000
# เปิด http://localhost:8000
```
